# 图遍历技能 (Graph Traversal Skill)

你具备使用 `neighbors` 工具来探索图结构的能力。

## 1. 何时使用 `neighbors`
寻找图中某个节点的相邻节点，或者被要求寻找最远节点时，必须调用 `neighbors` 工具。严禁不调用工具直接盲猜图的拓扑。

## 2. 如何从 A 出发逐层探索 (严格 BFS 队列机制)
你必须像一台无情的计算机一样执行广度优先搜索：
- **第一层：** 必须先查询起点 `A` 的邻居。
- **强制追溯：** 对于工具返回的**每一个**新节点，你**必须全部**去调用 `neighbors` 查询它们的下一层！
- **严禁偷懒：** 如果你查了 C，发现它有一个邻居是 F，你**必须**在下一步接着去查 F 的邻居！只要你的视野里出现了任何一个还没有查过邻居的新节点，就绝对不允许停止调用工具！

## 3. 如何避免重复访问
在脑海中维护一个“已访问列表”。如果 `neighbors` 工具返回的某个节点，你之前已经对它调用过工具了，就不要再查它，直接跳过，防止陷入死循环。

## 4. 最终答案输出格式
**警告：** 图的最深处藏得很深。必须直到所有发现的节点都查过了，且再也没有新节点产生时，才能停机！
当你彻底穷尽了整张图后，在最后一次回答的末尾，必须严格包含以下句子来输出答案（应包含最远节点 G）：
"The farthest node from A is G."
